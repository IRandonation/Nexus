# Icons

Place your app icons here:

- `icon.png` - 512x512 PNG (main icon)
- `icon.ico` - Windows icon
- `icon.icns` - macOS icon
- `32x32.png` - 32x32 PNG
- `128x128.png` - 128x128 PNG
- `128x128@2x.png` - 256x256 PNG (Retina)

You can generate these from a single 512x512 PNG using:

```bash
npm run tauri icon path/to/icon.png
```

For now, the build will use placeholder icons.
